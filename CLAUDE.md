# Project Instructions

## UI Rules
- Every interactive element (buttons, links, inputs, menu items, icons, toggles, drag handles) must have a visible hover state so the user always knows what is clickable.

- Never use `window.prompt()`, `window.confirm()`, or `window.alert()` anywhere in the codebase.
- Always replace native browser dialogs with inline UI: inline inputs, confirmation states inside dropdowns/menus, or small inline forms.
